## Описание

Приложение представляет собой интерактивный атлас точек акупунктуры, предназначенный для специалистов и практикующих электроакупунктуру. Оно позволяет:

- Просматривать расположение и описание точек по патологиям.
- Добавлять собственные схемы и точки, редактировать существующие.
- Вести историю сеансов с возможностью регистрации нескольких пользователей.
- Фиксировать показания приборов (тестер Фолля, Эледия) и анализировать динамику.
- Использовать секундомер для контроля времени воздействия на точку.
- Экспортировать и импортировать данные в формате CSV для обмена схемами и резервного копирования.

## Основные функции

### Главная страница
- Отображает список патологий и относящиеся к ним точки акупунктуры.
- При клике на название точки открывается карточка с подробным описанием и изображением.
- Секундомер позволяет отслеживать время проработки точки.
- Кнопка «Добавить точку» даёт возможность внести в базу новые точки или изменить существующие.

### Импорт / Экспорт CSV
- Базовый набор точек загружается из файла `point.json`.
- Все добавленные или изменённые данные сохраняются в локальном хранилище браузера.
- Экспорт в CSV сохраняет все точки (базовые + пользовательские) для резервного копирования или переноса на другое устройство.
- Импорт из CSV позволяет восстановить данные или объединить с текущими.

### Поиск и аналитика
- **Поиск по названию точки** — показывает все патологии, в которых встречается данная точка, что позволяет оценить её влияние на смежные системы организма.
- **Общие точки (суммационные пункты)** — выбор нескольких патологий показывает точки, общие для всех выбранных, что важно для комплексного лечения.

### История пользователя
- Возможность создавать нескольких пользователей и переключаться между ними.
- История сеансов группируется по датам (свернута по умолчанию), внутри — по патологиям.
- Для каждой точки можно ввести:
  - **Показания тестера Фолля** (значение до воздействия на точку эллектро аккупунктурой Эледия). Цвет рамки поля соответствует состоянию точки:
    - 0–20: чёрный
    - 21–28: фиолетовый
    - 29–38: синий
    - 39–48: индиго
    - 49–65: зелёный (оптимальный диапазон)
    - 66–80: оранжевый
    - 81–100: красный
  - **Показания прибора Эледия** (электроакупунктура).
- После нажатия кнопки «save» значения фиксируются и не подлежат изменению.
- При клике на название точки открывается её карточка с полным описанием.

### Хранение данных
- Все пользовательские данные (точки, история, измерения) хранятся в `localStorage` браузера.
- **Внимание:** очистка истории браузера может привести к потере данных. Рекомендуется регулярно экспортировать данные в CSV.
